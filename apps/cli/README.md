# Lere CLI

Command-line interface for extracting legal ROMs from various platforms.

## Installation

```bash
# Install dependencies
pnpm install

# Build the CLI
pnpm build
```

## Development

```bash
# Run in development mode with watch
pnpm dev
```

## Usage

```bash
# Show help
./dist/index.js

# Extract ROMs
./dist/index.js extract
```

## Available Commands

- `extract` - Extract ROMs from supported platforms
